A no_std ArtNet 4 implementation for creating microcontroller-based ArtNet Nodes.

## Implemented

✓ Node Discovery:
    ✓ ArtPoll: Parser
    ✓ ArtPollReply: Serialization
✓ DMX Lighting Control:
    ✓ ArtDMX: Parser
✓ Re-Programming:
    ✓ ArtCommand: Parser

## Not Implemented

Literally everything else. Pull Request are welcome.
